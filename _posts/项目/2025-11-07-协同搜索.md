# SDF-MAP

1. 将世界坐标转换成离散的体素索引

```cpp
inline void SDFMap::posToIndex(const Eigen::Vector3d &pos, Eigen::Vector3i &id)
{
    for (int i = 0; i < 3; ++i) id(i) = floor((pos(i) - mp_->map_origin_(i)) * mp_->resolution_inv_); // 除分辨率表示在单位距离内有多少体素
}
```
